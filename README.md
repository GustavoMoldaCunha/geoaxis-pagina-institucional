
# GeoAxis – Consultoria e Engenharia Ambiental

Site institucional desenvolvido para a **GeoAxis**, empresa especializada em consultoria e engenharia ambiental com atuação na Região Serrana e Metropolitana do Rio de Janeiro.

🔗 **[Ver site publicado](https://geoaxisambiental.com.br)**

--------------------------------------------------------------------------------------------------

## Sobre o projeto

O cliente precisava de uma presença digital profissional para apresentar seus serviços, transmitir credibilidade e facilitar o contato com potenciais clientes. O site foi desenvolvido com foco em clareza visual, responsividade e conversão — com acesso direto ao WhatsApp em todas as páginas.

--------------------------------------------------------------------------------------------------

## Tecnologias utilizadas

- **HTML5** semântico e acessível
- **CSS3** estilo personalizado
- **JavaScript** vanilla para interações (accordion, marquee)
- **Formspree** para envio de formulário de contato sem back-end
- **Font Awesome 5** para ícones
- **Netlify** para deploy e hospedagem

--------------------------------------------------------------------------------------------------

## Funcionalidades

- **Layout responsivo** adaptado para mobile, tablet e desktop
- **Animações de scroll** (fade-in nas seções ao rolar a página)
- **Carrossel marquee** infinito na seção "Por que escolher a GeoAxis?"
- **Accordion simples no botão "Ver mais"** nos cards de serviços com listas extensas
- **Botão flutuante** de WhatsApp
- **Formulário de contato** funcional integrado ao Formspree
- **Paleta de cores** personalizada alinhada à identidade visual do cliente

## Decisões de desenvolvimento

- **Template como base:** O HTML5 UP Story foi escolhido por sua estrutura de seções em tela cheia, ideal para apresentar serviços de forma impactante. As customizações foram feitas diretamente no `main.css` para manter o código organizado.

- **Sem framework:** O projeto foi desenvolvido sem frameworks CSS ou JavaScript para manter a simplicidade e performance, adequado ao perfil de um pequeno negócio.

- **Formulário sem backend:** O Formspree foi integrado para enviar os contatos diretamente ao e-mail do cliente, sem necessidade de servidor próprio.

- **WhatsApp como canal principal:** O botão flutuante e os links de contato priorizam o WhatsApp, canal preferido do cliente para atendimento.
